# Smart Delivery

Sistema web para la gestión de entregas inteligentes, con control de usuarios, empresas, pedidos y conductores. Incluye panel administrativo moderno y gestión de roles.

## Tecnologías utilizadas

- Python
- Django
- SQLite (desarrollo)
- Git & GitHub

## Funcionalidades

- Registro de usuarios con código de empresa
- Panel administrativo
- Gestión de empresas
- Gestión de conductores
- Dashboard con métricas
- Control de pedidos e incidencias

## Instalación

1. Clonar repositorio:
   git clone https://github.com/smartdelivery-2025/smart_delivery.git

2. Crear entorno virtual:
   python -m venv venv

3. Activar entorno:
   venv\Scripts\activate

4. Instalar dependencias:
   pip install -r requirements.txt

5. Migraciones:
   python manage.py migrate

6. Ejecutar:
   python manage.py runserver
