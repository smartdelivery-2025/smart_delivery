Smart Delivery

Sistema backend desarrollado con Django para la gestión de usuarios y administración.

# Tecnologías utilizadas

- Python
- Django
- SQLite
- Git & GitHub

# Instalación

1. Clonar repositorio:
   git clone https://github.com/karolina-dev/smart_delivery.git

2. Crear entorno virtual:
   python -m venv venv

3. Activar entorno:
   venv\Scripts\activate

4. Instalar dependencias:
   pip install -r requirements.txt

5. Ejecutar migraciones:
   python manage.py migrate

6. Ejecutar servidor:
   python manage.py runserver
